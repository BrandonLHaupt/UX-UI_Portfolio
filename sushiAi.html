Overview

Sushi Ai sells sushi for a decent price, and you can get an all-you-can-eat sushi, appetizer, and more.

Roles

UI Designer Interaction design, Prototyping, Wireframing, Layout Design, Design Systems

Goals

My goal is to create a more modern design and allow for customers to find what they are looking for more easily.

Background

During this project I used the double diamond process to efficiently go through define what the users are having issues with, conduct research, and create a solution for these issues

Personas

Problems: Users are having a hard time with the old site navigations, finding what they want to eat, looking for pictures, and messy layout. It is important to create and show the persona so that we can get into the user's mindset and make it a user-centered design.
